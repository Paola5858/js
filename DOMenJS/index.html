// 01 — querySelector · querySelectorAll · getElementById · getElementsByClassName · getElementsByTagName

querySelector

Selecionar pela tag

Pega o primeiro elemento que bater com o seletor. Funciona igual CSS: tag, .classe, #id.

let selecinarPorTag =
  document.querySelector("h1");

selecinarPorTag.textContent =
  "Novo texto 1";

querySelectorAll

Selecionar todos os p

Retorna uma NodeList, que funciona como array. Você acessa cada item pelo índice: [0], [1]...

let todasAsTags =
  document.querySelectorAll("p");

// índice [0] = primeiro p
todasAsTags[0].textContent =
  "Pagragráfo 1";
todasAsTags[1].textContent =
  "Pagragráfo 2";

ClassName · TagName

Classe e Tag

Ambos retornam HTMLCollection (como array). Acesse por índice também.

// pela classe
let porClasse =
  document.getElementsByClassName(
    "subtitulo");
porClasse[0].textContent =
  "Novo parágrafo";

// pela tag
let porTag =
  document.getElementsByTagName(
    "h2");
porTag[0].textContent =
  "Novo Paragrafo 3";

innerHTML aceita HTML dentro da string. textContent só texto puro. Use innerHTML quando quiser formatar.

let titulo =
  document.querySelector("#titulo");

titulo.innerHTML =

Mudar atributos

element.style

Mudar CSS via JS

Propriedades com hífen viram camelCase: background-color → backgroundColor. É a regra de ouro.

// cor do texto
titulo.style.color = "red";

// nome composto: camelCase
titulo.style.backgroundColor =
  "black";

setAttribute · removeAttribute

Setar e remover classe CSS

Você pode trocar a classe de um elemento pelo JS, aplicando qualquer estilo do seu CSS. E remover quando quiser.

let container =
  document.querySelector(".container");

// aplica a classe .amarelo do CSS
container.setAttribute(
  "class", "amarelo");

// remove o atributo class
container.removeAttribute("class");

// 03 — as 3 partes do DOM: seleção · manipulação · evento

O que é cada parte?

Todo código DOM pode ser dividido em 3 responsabilidades. O exemplo abaixo (exercício do professor) usa as três ao mesmo tempo.

① Seleção

Encontrar o elemento na página. Usa querySelector, getElementById etc. Sem seleção, você não tem o elemento para trabalhar.

② Manipulação

Alterar o elemento já selecionado: mudar texto (textContent), cor, tamanho, atributos. É o que de fato muda o que o usuário vê.

③ Evento

Escutar uma ação do usuário (clique, tecla, mouse...) com addEventListener. A manipulação só acontece quando o evento dispara.

Código do exercício

As 3 partes no mesmo código

Cada linha colorida pertence a uma das três responsabilidades. Veja como elas se encadeiam.

// ① SELEÇÃO — acha os elementos
const meuTitulo = document.querySelector('#titulo');
const meuBotao  = document.querySelector('#btnMudar');

// ③ EVENTO — escuta o clique no botão
meuBotao.addEventListener('click', () => {

  // ② MANIPULAÇÃO — altera o título
  meuTitulo.textContent  = "O DOM FUNCIONOU!";
  meuTitulo.style.color    = "red";
  meuTitulo.style.fontSize = "50px";

  console.log("botão clicado!");
});

// 04 — DOM 2: textContent + style · DOM 3: classList.toggle

DOM 2

Mudar título com clique

① Seleção

Encontra os elementos da página pelo ID.

② Manipulação

JS muda texto, cor e tamanho depois do clique.

③ Evento

Clicar no botão dispara o addEventListener('click').

// ① SELEÇÃO
const meuTitulo = document.querySelector('#titulo');
const meuBotao  = document.querySelector('#btnMudar');

// ③ EVENTO
meuBotao.addEventListener('click', () => {

  // ② MANIPULAÇÃO
  meuTitulo.textContent  = "O DOM FUNCIONOU!";
  meuTitulo.style.color    = "red";

DOM 3

Modo escuro com classList.toggle

① Seleção

Pega o corpo da página e o botão para poder usar depois.

② Manipulação

Liga ou desliga o modo escuro — adiciona ou remove a classe modo-escuro.

③ Evento

Fica "escutando" o botão. Quando o usuário clicar, executa o código.

// ① SELEÇÃO
const corpoPagina = document.body;
const botaoTema   = document.querySelector("#btnTema");

// ③ EVENTO
botaoTema.addEventListener('click', () => {

  // ② MANIPULAÇÃO
  corpoPagina.classList.toggle('modo-escuro');
});
