// 01 — querySelector · querySelectorAll · getElementById · getElementsByClassName · getElementsByTagName

querySelector

Selecionar pela tag

Pega o primeiro elemento que bater com o seletor. Funciona igual CSS: tag, .classe, #id.

let selecinarPorTag =
  document.querySelector("h1");

selecinarPorTag.textContent =
  "Novo texto 1";

querySelectorAll

Selecionar todos os p

Retorna uma NodeList, que funciona como array. Você acessa cada item pelo índice: [0], [1]...

let todasAsTags =
  document.querySelectorAll("p");

// índice [0] = primeiro p
todasAsTags[0].textContent =
  "Pagragráfo 1";
todasAsTags[1].textContent =
  "Pagragráfo 2";
resultado ao vivo

getElementById

Selecionar por ID

O mais direto. ID é único na página, então sempre retorna um único elemento, nunca array.

let selecionarPorId =
  document.getElementById("txt2");

selecionarPorId.textContent =
  "Novo texto 2";

ClassName · TagName

Classe e Tag

Ambos retornam HTMLCollection (como array). Acesse por índice também.

// pela classe
let porClasse =
  document.getElementsByClassName(
    "subtitulo");
porClasse[0].textContent =
  "Novo parágrafo";

// pela tag
let porTag =
  document.getElementsByTagName(
    "h2");
porTag[0].textContent =
  "Novo Paragrafo 3";

// 02 — innerHTML · setAttribute · style · classList

innerHTML / textContent

Mudar o conteúdo

innerHTML aceita HTML dentro da string. textContent só texto puro. Use innerHTML quando quiser formatar.

let titulo =
  document.querySelector("#titulo");

titulo.innerHTML =
  "Coloquei o novo título";

setAttribute

Mudar atributos

Serve pra qualquer atributo HTML: src, href, class, id, width... Você passa o nome do atributo e o novo valor.

let imagem =
  document.querySelector("#foto");

imagem.setAttribute(
  "src",
  "https://picsum.photos/200/200");

// imagem.setAttribute("width","400");

element.style

Mudar CSS via JS

Propriedades com hífen viram camelCase: background-color → backgroundColor. É a regra de ouro.

// cor do texto
titulo.style.color = "red";

// nome composto: camelCase
titulo.style.backgroundColor =
  "black";

setAttribute · removeAttribute

Setar e remover classe CSS

Você pode trocar a classe de um elemento pelo JS, aplicando qualquer estilo do seu CSS. E remover quando quiser.

let container =
  document.querySelector(".container");

// aplica a classe .amarelo do CSS
container.setAttribute(
  "class", "amarelo");

// remove o atributo class
container.removeAttribute("class");
